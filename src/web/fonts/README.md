### What

This folder contains FontAwesome 7 Free icon fonts.

The fonts are "subset" meaning they only include the icons we need, to reduce file size.

The fonts and the all.min.css stylesheet are generated by `fontawesome-subset.mjs` tool
in src/vite/ (see the comments in the file for more detail on how to run it).

The subset fonts are included in the public repo now because they are created from the
*free* version of FontAwesome 7.

### Context

As a "Font Awesome 5 Early Backer" I can still download the 5.15.4 pro icons and could
use them in the app in SVG format. But for now we use the font version of the icons.
Since my FA5 Pro token was disabled, I can no longer import the Pro font packages and I can
not download the Pro subsetter tool. So I figured we can do with the Free icons anyway,
this allows to use the `fontawesome-subset` tool and the FontAwesome 7 Free package to
create a subset version of the font.
